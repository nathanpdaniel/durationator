# **App Name**: Durationator

## Core Features:

- Duration Input: Allow users to input a list of durations (hours and minutes).
- Total Duration Calculation: Calculate the total duration from the list of input durations.
- Target Duration Input: Provide an input field for users to specify their target duration.
- Remaining Duration Calculation: Calculate and display the remaining duration until the target duration is reached.
- User Guide Tool: Use a tool to suggest explanations and information to assist users of this app, in response to user prompts.

## Style Guidelines:

- Primary color: Dark Blue (#3F51B5) to evoke a sense of focus and precision.
- Background color: Very light grayish-blue (#E8EAF6).
- Accent color: Orange (#FF9800) to highlight key information and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Use simple, clear icons to represent actions like adding durations or resetting the timer.
- Arrange the duration input fields and results in a clear, linear layout for ease of use.
- Provide smooth transitions when calculating and displaying the remaining time.
